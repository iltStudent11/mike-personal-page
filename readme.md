This is my readme.md file for my personal page.

You can access my project here: https://github.com/iltStudent11/mike-personal-page.git

I create this personal page to practice what I learned in the HTML training videos.  Here I will start by creating a index.html 'home page' as a starting point.  I will also add a styles.css to my project.

I created a features branch so that I could practice with my index.html file without worrying about breaking anything on the main branch.  This allows me to try new things and practice new things in a sandbox type environment.

My commits are based off of completing a function.  Creating a table, adding an image, or completing the structure of my file.  I use my commit messages to explain why I did what I did, and not how I did it.  I want to keep my commit comments short and informational, so that other users can quickly see why I made that commit.

I am going to start with an index.html file and expand from there.  Just like the training iceos, I plan to add a contact.html file and a styles.css file.
